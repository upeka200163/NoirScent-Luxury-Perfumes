# 🖤 NoirScent – Luxury Perfume E-Commerce WebSite

**NoirScent** is a high-end, fully responsive e-commerce landing page crafted for a luxury fragrance brand.
This project blends a **dark luxury aesthetic** with modern front-end development to deliver a visually rich and seamless user experience.

---

## ✨ Overview

NoirScent is designed to reflect elegance, exclusivity, and premium branding.
From immersive visuals to smooth interactions, every element is carefully built to enhance user engagement and conversion.

## 🌟 Key Features

### 🎯 Premium Hero Section

* Immersive full-width background
* Strong call-to-action buttons: **Shop Now** & **Explore Collections**

### 🛍️ Curated Product Categories

* Men, Women, Oud Collections, Gift Sets
* Elegant hover effects for better interaction

### 💎 Best Sellers Showcase

* Interactive product grid

* Includes pricing and star ratings
